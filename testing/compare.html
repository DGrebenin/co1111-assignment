Testing API

Actual API

Lists

For the testing List, the user can enter a specific number of T.H lists that wants to access. In the actual game, the lists are automatically shown up by the Server.

Questions

In the testing Questions, the user can choose which type of questions wants to access. In the actual game, the questions will be shown up in the screen, depending the type.

Answers

In the testing Answer, user can choose to see the response if the answer is True/False. In the actual Game the answer is shown up on the screen depending what the user answered.

Leaderboard

For the testing Leaderboard the user enters a specific numbers of users that will be printed on the table. The user can choose either the table will be sorted or if the player has Prize. In the actual Leaderboard the user can choose the size of the array setting the limit on the URL. Moreover, the user can choose if the table will be printed sorted or not by typing 'Sorted' || 'Unsorted' in the URL
